# Pagi Sore Catering - Front-end

Simple front-end scaffold for Pagi Sore Catering System.

Features included:
- Customer pages: register, login, menu, order form, order summary, upload payment, status tracking, profile, dashboard.
- Admin pages: admin login, dashboard, menu management, payment validation, orders list, delivery management.

Tech: plain HTML, CSS, and JavaScript. Dummy endpoints are in `js/api.js`.

Run: Open `index.html` in your browser (static files). No server required for initial testing.
